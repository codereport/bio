# Conor Hoekstra - Linktree / About Me

A simple, responsive static site for Conor Hoekstra's profile, featuring:
- Social media links (Twitter, GitHub, YouTube, LinkedIn, Mastodon, Bluesky)
- Podcast links
- Resources link

## Setup

No build process required. This is a static HTML/CSS/JS project.

1. Open `index.html` in your browser to view.
2. To deploy, simply host the files (`index.html`, `styles.css`, `script.js`) on any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Customization

- **Icons**: Uses FontAwesome 6.6.0 via CDN.
- **Theme**: Colors are defined in `styles.css` under `:root` and `[data-theme="dark"]`.
